# toolbox

#### 介绍
一个工具箱。

#### 软件架构
软件架构说明

前端：electron + vite +vue3 +ts
后端：springboot + springData

#### 安装教程

1.  安装依赖

   ```
   pnpm install
   ```

   

#### 使用说明
​	推荐使用pnpm安装：

​    换源命令 : 

```
npm set registry https://registry.npmmirror.com
```

1.  pnpm dev 启动web页面
2.  pnpm electron:dev 启动桌面页面
3.  pnpm build 打包桌面app
4.  render:build 打包web页面
